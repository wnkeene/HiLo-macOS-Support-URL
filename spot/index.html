    <div class="content">

        <h2>About the App</h2>

        <p>Spot Calculator is a precision tool for agricultural spray mixing calculations. Calculate exact amounts for tank mixing using either the label rate method or percentage solution method.</p>


        <h2>Features</h2>

        <ul>

            <li>Imperial and Metric unit systems</li>

            <li>Label Rate/Acre and Label Rate/Hectare calculations</li>

            <li>Percentage Solution mode</li>

            <li>Multiple unit options (oz, lb, pt, qt, gal / mL, L, g, kg)</li>

            <li>Precise decimal calculations</li>

            <li>Clean, intuitive Apple-style interface</li>

            <li>Works on iPhone, iPad, and Mac (Catalyst)</li>

        </ul>


        <h2>How to Use</h2>

        <ol>

            <li>Select Imperial or Metric unit system</li>

            <li>Choose calculation mode: % Solution or Rate/Acre (or Rate/Hectare)</li>

            <li>Select unit of measurement for label amount</li>

            <li>Enter label rate or percentage</li>

            <li>Enter tank size</li>

            <li>Tap "Compute" to see result</li>

        </ol>
