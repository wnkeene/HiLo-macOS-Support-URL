Spot Calculator

Agricultural Spray Calculator for iOS and iPadOS

About the App

Spot Calculator is a precision tool for agricultural spray mixing calculations. Calculate exact amounts for tank mixing using either the label rate method or percentage solution method.

Features

  • Imperial and Metric unit systems
  • Label Rate/Acre and Label Rate/Hectare calculations
  • Percentage Solution mode
  • Multiple unit options (oz, lb, pt, qt, gal / mL, L, g, kg)
  • Precise decimal calculations
  • Clean, intuitive Apple-style interface
  • Works on iPhone, iPad, and Mac (Catalyst)

How to Use

  1. Select Imperial or Metric button and unit of measurement button for amount given on product label.
  2. Press either % Solution or Rate/Acre (Hectare for metric) button to use with label method.
  3. Enter Tank Size in digets only. Units of measurement will be pre-set for you from previous steps.
  4. Tap "=" to see the amount to add to tank.
